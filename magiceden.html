Тест открытия kfcgovno.cyou в Magic Eden Wallet

Важно: Magic Eden Wallet не имеет публичной deep link схемы для открытия URL (типа magiceden://browse/...).

Если приложение установлено, попробуйте кнопки ниже — одна из них может сработать (fallback на обычный браузер).

Лучший способ: откройте Magic Eden app → вкладка Browser → введите адрес вручную.




Тестируйте на мобильном устройстве с установленным Magic Eden Wallet (из App Store / Google Play).

Если ничего не работает — это нормально, потому что официально deep link для внешнего открытия dApp не документирован.
